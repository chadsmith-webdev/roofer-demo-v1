# Summit Ridge Roofing & Restoration - NWA Demo Site

A premium, conversion-optimized multi-page roofing contractor website built with the **StoryBrand (SB7)** framework, **Marketing Psychology**, and **Local SEO** best practices.

## 🚀 Overview

This site is a "Pro Max" demo built for a fictional roofing business in **Fayetteville, Arkansas**. It is designed to serve as a high-end product for local search and web development agencies targeting the roofing niche.

### Key Features

- **StoryBrand Messaging**: 
    - **Hero**: Homeowners/Business owners in NWA.
    - **Villain**: Dishonest "storm chaser" contractors.
    - **Guide**: Summit Ridge (Authority + Empathy).
    - **The Scales (Stakes)**: Contrast between Success and Failure.
    - **The Plan**: Clear 3-step visualization to resolve client anxiety.
- **UI/UX Pro Max Design**:
    - **Glassmorphism**: Modern, frosted-glass components using CSS backdrop-filter.
    - **Dark Mode**: Sleek Charcoal/Slate palette with Electric Blue accents.
    - **Micro-animations**: Scroll-triggered reveal effects using Intersection Observer.
    - **Interactive Visualizer**: A concept-driven area for visualizing material choices.
- **Local SEO & Geo-Targeting**:
    - Focus on **Fayetteville, Springdale, Rogers, and Bentonville**.
    - **Schema.org** JSON-LD implemented for LocalBusiness and RoofingContractor.
    - **Humanized Content**: Rugged local craftsmanship tone instead of robotic "corporate-speak."

## 🛠️ Tech Stack

- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3**: Custom design system (vanilla CSS) with modern layout and glass effects.
- **JavaScript**: Optimized interactivity and scroll-reveal logic.
- **Images**: AI-generated high-fidelity assets for a professional, branded look.

## 📁 Directory Structure

- `index.html`: StoryBrand Homepage.
- `services.html`: Service silos for SEO authority.
- `about.html`: Brand story and "Guide" bio.
- `contact.html`: High-conversion lead funnel.
- `css/index.css`: Global design system and utilities.
- `js/script.js`: Interactivity and animation logic.
- `images/`: Custom-generated project and hero assets.
- `schema.json`: Structured data for Google Search.

## 💡 How to Use This Demo

1. Open `index.html` in any modern web browser.
2. Experience the scroll-reveal transitions as you move through the StoryBrand narrative.
3. Test the "Free Inspection" lead capture form on the contact page.
4. Review the Service Area silos in the footer for Local SEO context.
